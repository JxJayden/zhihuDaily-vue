# zhihu-daily-vue

> zhihu-daily-vue 是使用 vue2.0 vue-router vue-resource node.js 开发的知乎日报

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run api server
npm run server
```
